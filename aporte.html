Conocimientos Back-end

  • He utilizado diferentes patrones de diseño como lo son Mediator, Singleton, Proxy, Decorator, entre otros, en proyectos que he realizado.
  • Sé usar el patrón MVC y además tengo cimientos en DDD, una arquitectura limpia que favorece el código limpio y desacoplado.
  • Sé utilizar los principios SOLID a la hora de escribir código, esto con la intención de tener un estándar más alto en cuanto a la calidad de la aplicación.
  • Desarrollo de REST API de la mano con Postman.
  • Documentación de la API con Postman.
  • Uso de GIT y de GitHub en cada uno de los proyectos que he realizado.
  • Creación y administración de una Base de Datos.
  • Consultas SQL nativas, así como también con el ORM de Laravel.
  • Uso de las herramientas que proporciona Laravel como Framework.

Conocimientos Front-end

  • Sé utilizar el stack básico de Front-end: HTML, CSS y Javascript.
  • He utilizado AJAX y la librería Axios con el fin de hacer peticiones HTTP sin necesidad de refrescar la página.
  • Tengo conocimiento básico en Vuejs, Framework de Javascript.
  • Dispongo de buen conocimiento para con el uso de Bootstrap.
